# Example

To run this example, clone the repository:

```bash
git clone https://github.com/purplnay/easy-api-test.git
```

`cd` to the repository folder:

```bash
cd easy-api-test
```

Build the library:

```bash
npm run build
```

`cd` to the http example directory:

```bash
cd examples/http
```

Install the dependencies:

```bash
npm install
```

Run the tests:

```bash
npm test
```
